Installation:
This program is made in python so you must first have python installed.

Then you must install the library used by running this command in your command prompt
pip install psycopg2

Then you must edit the code to ensure that the connect_db() function has the right information for your database, incase your password is different or something

once the library is installed you can run the program, there are example function calls commented out below
